The project aims to determine the potential risk of an application with the permission and APIs it asks.There are various permissions (features) with which we detected the malware.
The project comprises of various machine learning models including Logistic Regression, Naive Bayes, ANN, KNN, SVM, Random forest Classifier, using 3 feature selection methods.
The accuracy of each model with the feature selection methods are compared to find the best model and feature combination.
